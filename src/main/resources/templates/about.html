Smart Contact Manager is a secure and user-friendly web application designed to help individuals and organizations manage their personal and professional contacts efficiently

✨Key Features

🔐 Secure Authentication & Authorization using Spring Security
📇 Add, Edit, Delete Contacts with full CRUD functionality
📁 File Upload Support for contact profile images
📱 Responsive UI using Thymeleaf templates
🔍 Search Contacts by name or email
🧾 Role-Based Access (Admin/User)
💾 Database Integration with MySQL
📊 Dashboard Overview of contacts

💻Tech Stack

Backend:
Java, Spring Boot, Spring MVC, Spring Security, Hibernate (JPA), MySQL
Frontend:
HTML, CSS, Bootstrap, Thymeleaf
